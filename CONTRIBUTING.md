# Contributing to Advanced Expense Manager 💸

Thank you for your interest in the Advanced Expense Manager! This project has evolved from a basic tracker into an AI-powered financial system, and we welcome contributions to keep improving it.

## 🌟 How Can I Help?

### 1. Improvements and Features
*   **AI Insights**: Suggesting new prompts or ways to analyze spending behavior.
*   **Backend Optimization**: Improving the Node.js / Express.js performance.
*   **UI/UX**: Refining the Android interface and data visualization.

### 2. Reporting Issues
Please provide detailed steps to reproduce any bugs you find.

### 3. Contribution Rules
*   Please keep the code documented and clean.
*   Ensure the backend and frontend remain synchronized.
*   Do not push raw `.env` files or API keys.

## 🤝 Project Origin
This project was originally based on a foundational offline tracker. It has since been significantly expanded with a full-stack architecture, JWT authentication, and AI-driven insights. We respect the original work that served as the starting point for this system.
